# Masater.tj.ff

Репозиторий проекта **Master.tj** — маркетплейс услуг для клиентов и мастеров.

## Содержимое

| Папка / файл | Описание |
|--------------|----------|
| `masterchas_app/` | Flutter-приложение (основной код) |
| `masterchas_app/README.md` | **Полная документация проекта** |
| `run.sh` / `run.bat` | Быстрый запуск приложения из корня |

## Быстрый старт

```bash
cd masterchas_app
flutter pub get
./run.sh
```

Подробное описание экранов, файлов, маршрутов и архитектуры — в [masterchas_app/README.md](masterchas_app/README.md).
